## ToDo App - Aplicativo de Gerenciamento de Tarefas


 ###  Sobre o Projeto
Este é um aplicativo web de gerenciamento de tarefas (ToDo List) desenvolvido com tecnologias front-end modernas. O projeto permite aos usuários criar, visualizar e gerenciar suas tarefas de forma intuitiva e organizada.

### Funcionalidades
 - Adicionar novas tarefas

 - Visualizar progresso de execução

 - Categorizar tarefas (importantes, finalizadas, não finalizadas)

 - Sistema de busca de tarefas

 - Definição de prazos para tarefas

 - Armazenamento local dos dados

 - Interface responsiva e moderna

### Tecnologias Utilizadas
 - HTML5 - Estruturação do conteúdo

 - Tailwind CSS - Framework CSS para estilização

 - JavaScript - Funcionalidades e interatividade

 - Font Awesome - Ícones e elementos visuais: Link: https://fontawesome.com/

 - Local Storage API - Armazenamento local de dados

### Características Técnicas
 - Design System: Interface com cores escuras e tons de roxo (#6D28D9) para melhor experiência visual

 - Layout: Divisão em sidebar e área de conteúdo principal

 - Responsividade: Design adaptável para diferentes tamanhos de tela

 - Modal Interativo: Sistema de pop-up para adição de tarefas

 - Cards Dinâmicos: Visualização em grid das tarefas cadastradas

### Como Utilizar
1. Clone o repositório

2. Abra o arquivo index.html em seu navegador

3. Clique em "Adicionar Tarefa" para criar novas atividades

4. Preencha os dados da tarefa (nome, descrição, prazo)

5. Marque como importante se necessário

5. Visualize suas tarefas organizadas em cards

### Funcionalidades JavaScript
 - Gerenciamento de modais (abrir/fechar)

 - Captura de dados do formulário

 - Armazenamento local usando LocalStorage

 - Manipulação do DOM para exibição dinâmica
